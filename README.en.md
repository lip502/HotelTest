# 基于javaweb在线点餐系统

#### Description
前台涉及的功能：餐桌的占位与取消占位，购物车功能，下单功能，菜单管理，订单管理，用户登录及记住密码，用户注册。

后台涉及的功能：餐桌管理，菜品类型管理，菜品管理，订单管理，用户登录及记住密码，用户注册。
